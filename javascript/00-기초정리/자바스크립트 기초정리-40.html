자바스크립트 객체 핵심요약

1. 객체의 형태부터 구분할 수 있어야 한다. - {....}

2. 객체에 저장하는 데이터는 대게 서로 연관성을 가진 경우가 많다.

3. 값을 꺼내오는 방법 - 객체변수명.속성명

4. 객체변수는 속성값으로 함수도 올(할당할) 수 있다.

        const mycar = {
            brand : "현대",
            name : "펠리셰이드",
            color : "black",
            price : 50000000,

            drive : function(){alert('갈릉으로 출발~!');}
        }

다른 문법(방법)으로 객체변수 만들기

        const mycar = new Object();
        
        mycar.brand = "현대";
        mycar.name = "펠리셰이드";
        mycar.color = "black";
        mycar.price = 50000000;
        mycar.drive = function(){alert('갈릉으로 출발~!');};
