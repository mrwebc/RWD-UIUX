대소문자 구별해서 작성

변수에 대해서는 이것만 알면 된다.

DOM을 셀렉팅하는 변수명
관례적으로 $로 시작한다.
const 키워드는 상수선언시 사용
중간에 상태값을 변경할 수 없다.
let 키워드는 변수선언시 사용
중간에 상태값 변경(재할당)이 가능하다.
변수명 작성시 특수문자 사용 불가
특별히 $, _(언더바) 만 사용 가능

연산자에 핵심정리

+ 기호가 ★연결연산자★ 로 인식되는 경우
const name = "김수한" + "무거북이와두루미";
let money = 100 + "50" + "원";
